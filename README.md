
# Vivre de la mer à Geoje

Documentaire interactif web autour de la pêche rurale en Corée du Sud, plus précisément sur l’île de Geoje.

Le projet propose une expérience immersive mêlant vidéo documentaire, navigation interactive, contenus informatifs et exploration visuelle. L’objectif est de faire découvrir la richesse, la fragilité et les mutations des communautés de pêcheurs face à l’industrialisation, ...
## 🌊 Concept du projet

**Vivre de la mer à Geoje** est un documentaire interactif qui invite le spectateur à explorer le quotidien de personnes liées au monde maritime :

- le pêcheur ;
- le poissonnier ;
- le restaurateur.
